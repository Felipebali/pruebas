Sobre mi Servidor Local

¡Hola! Soy Feli, y esto es mi servidor local de pruebas. Aquí puedo experimentar con HTML, CSS y JavaScript para crear páginas interactivas y dinámicas. Cada página tiene funcionalidades como cambio de fondo, reloj en tiempo real, listas de tareas y mucho más.

El objetivo de este servidor es aprender y probar ideas, desde páginas estáticas hasta interactividad avanzada. ¡Podés explorar todas las funcionalidades y divertirte mientras aprendes!
